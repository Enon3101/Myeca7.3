# MyeCA.in - Website Overview

MyeCA.in is a premium, AI-powered financial and tax services platform designed for individuals, startups, and businesses in India. It simplifies complex compliance tasks through automation, expert CA review, and intuitive tools.

## 🚀 Key Features

### 1. ITR Filing (Income Tax Return)
- **Fastest Filing**: 4-step process to file ITR online.
- **CA Verified**: Every return is manually reviewed by a certified Chartered Accountant.
- **Form 16 Parser**: Auto-extract data from Form 16 and bank statements using AI.
- **Maximum Refund Guarantee**: Expert optimization for zero-error filing.

### 2. Tax & Financial Calculators
- **Income Tax Calculator**: Supports New vs. Old regime analysis for AY 2025-26.
- **HRA Calculator**: Optimizes House Rent Allowance exemptions.
- **SIP & Investment Tools**: Mutual fund planning and retirement (NPS) calculators.

### 3. Startup & Business Services
- **Entity Registration**: Private Limited, LLP, OPC incorporation.
- **Compliance Hub**: GST returns, TDS filing, and MSME registrations.
- **Strategic Advisory**: Virtual CFO and funding assistance for startups.

### 4. AI & Advanced Tools
- **AI Tax Assistant**: Instant answers for tax queries and document analysis.
- **Tax Loss Harvesting**: Smart algorithms to save on capital gains tax.
- **Secure Document Vault**: Bank-grade storage for financial records.

## 🛠 Tech Stack

- **Frontend**: React (Vite), TypeScript, Tailwind CSS, Framer Motion (for animations).
- **Backend**: Node.js, Express.
- **Database**: PostgreSQL (Neon/Supabase) with Drizzle ORM.
- **Authentication**: Clerk (managed authentication).
- **AI Services**: OpenAI API Integration.
- **Email/Communications**: AWS SES & SendGrid.
- **Infrastructure**: Firebase (Service Integration).

## 📂 Project Structure

- `/client`: Frontend application code.
  - `/src/pages`: Individual feature pages (Calculators, Services, Dashboard).
  - `/src/components`: Reusable UI components and layout elements.
  - `/src/lib`: Logic utilities and API configurations.
- `/server`: Backend API and server-side logic.
- `/shared`: Shared TypeScript types and database schemas.
- `/supabase`: Configuration for Supabase services.

## 📈 Performance & Quality
